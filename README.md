# OpenClaw 自媒体技能合集

> 覆盖自媒体起号/变现全流程的28个OpenClaw技能
> 
> 🌐 **更多资源请访问：[nexaclaw.cn](https://nexaclaw.cn)**
> 
> 为内容创作者提供从选题、创作、发布到数据分析、商业变现的全套自动化工具。

---

## 📦 技能清单

### 1️⃣ 内容策划与选题
| 技能 | 说明 |
|------|------|
| `topic-generator` | 基于热点追踪和用户画像的内容选题生成 |
| `audience-analyzer` | 用户画像与受众分析工具 |
| `content-calendar` | 内容日历管理，发布计划与热点提醒 |

### 2️⃣ 文案创作
| 技能 | 说明 |
|------|------|
| `copywriter` | 多风格文案写作：种草/测评/品牌故事 |
| `headline-optimizer` | 高点击率标题生成与A/B测试 |
| `seo-writer` | SEO长文写作与关键词布局 |
| `script-writer` | 短视频脚本写作（口播/剧情/测评/vlog） |

### 3️⃣ 图片与视觉设计
| 技能 | 说明 |
|------|------|
| `canghe-xhs-images` | 小红书风格图文排版 |
| `canghe-cover-image` | 吸睛封面图生成 |
| `canghe-infographic` | 数据可视化信息图生成 |
| `meme-generator` | 热门表情包/梗图生成 |

### 4️⃣ 视频制作
| 技能 | 说明 |
|------|------|
| `video-editor` | 视频剪辑建议与节奏优化 |
| `script-writer` | 短视频脚本写作 |

### 5️⃣ 多平台发布
| 技能 | 说明 |
|------|------|
| `multi-post` | 多平台内容一键分发 |

### 6️⃣ 数据分析与复盘
| 技能 | 说明 |
|------|------|
| `content-performance` | 单篇内容深度分析与爆款拆解 |
| `follower-growth` | 粉丝增长趋势与来源分析 |
| `competitor-analysis` | 竞品数据监控与策略分析 |

### 7️⃣ 用户互动与社群运营
| 技能 | 说明 |
|------|------|
| `comment-manager` | 多平台评论聚合与智能回复 |
| `community-bot` | 社群自动化运营 |
| `dm-automation` | 私信自动回复与粉丝分层 |
| `social-listen` | 社交监听与舆情监控 |
| `feedback-collector` | 用户反馈收集与满意度调研 |

### 8️⃣ 商业变现
| 技能 | 说明 |
|------|------|
| `brand-match` | 品牌合作匹配与报价建议 |
| `affiliate-tracker` | 带货链接追踪与佣金统计 |
| `course-builder` | 知识付费课程规划与制作 |
| `shop-integration` | 平台小店商品与订单管理 |

### 9️⃣ 效率工具
| 技能 | 说明 |
|------|------|
| `asset-manager` | 图文/视频/文案素材库管理 |
| `collaboration-hub` | 团队任务分配与进度追踪 |

## 🚀 快速安装

### 克隆本仓库安装
```bash
git clone https://github.com/nexaclaw/openclaw-media-skills.git
cd openclaw-media-skills
# 链接需要的技能到 OpenClaw
ln -sfn $(pwd)/copywriter ~/.openclaw/skills/copywriter
```
