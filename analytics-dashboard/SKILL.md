# Analytics Dashboard - 数据分析仪表盘

## 触发场景

- "数据报表"
- "周报生成"
- "运营数据分析"
- "增长趋势"

## 支持平台

- 微信公众号
- 抖音
- 小红书
- B站
- 微博
- 知乎

## 功能特性

### 1. 数据聚合
- 多平台数据统一
- 自动抓取更新
- 历史数据存储

### 2. 可视化报表
- 增长趋势图
- 内容效果对比
- 粉丝画像
- 收益分析

### 3. 智能分析
- 爆款识别
- 内容建议
- 发布时间优化
- 竞品对比

### 4. 定时报告
- 日报/周报/月报
- 微信/邮件推送
- PDF导出

## 报表模板

```markdown
## 📊 运营周报

### 本周概览
- 新增粉丝：+XXX
- 总阅读：XXX
- 互动率：XX%

### 内容表现 TOP3
1. 标题1 - 阅读 XXX
2. 标题2 - 阅读 XXX
3. 标题3 - 阅读 XXX

### 增长趋势
[趋势图]

### 建议
- ...
```

## 使用方法

1. 连接各平台账号
2. 选择分析周期
3. 生成报表
4. 导出/推送
