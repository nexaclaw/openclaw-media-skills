# Asset Manager - 素材管理

## 触发场景

- "帮我管理素材"
- "上次那个图片在哪"
- "素材库整理"
- "批量处理素材"

## 功能模块

### 1. 素材分类
- 按内容类型：图文/视频/音频/文案
- 按主题分类：产品/教程/案例/热点
- 按平台分类：小红书/抖音/公众号
- 按时间分类：2024/2023/历史

### 2. 标签管理
- 多维度标签
- 自动标签建议
- 标签批量添加
- 标签搜索优化

### 3. 快速检索
- 关键词搜索
- 标签组合搜索
- 日期范围搜索
- 智能联想

### 4. 素材处理
- 图片压缩/裁剪
- 视频转码
- 批量重命名
- 格式转换

### 5. 素材统计
- 素材数量统计
- 使用情况追踪
- 热门素材排行
- 存储空间管理

## 目录结构建议

```
素材库/
├── 图片/
│   ├── 产品图/
│   ├── 场景图/

│   └── 表情包/
├── 视频/
│   ├── 原创/

│   └── 素材/
├── 文案/
│   ├── 标题库/
│   ├── 开头库/
│   ├── 结尾库/
│   └── 模板库/
└── 参考/
    ├── 竞品/
    └── 行业/
```

## 使用方法

1. 初始化素材库结构
2. 导入现有素材
3. 设置分类规则
4. 开始使用检索

示例：
"帮我把桌面上的图片按类型整理到素材库"
"搜索标签包含'春季'和'穿搭'的图片"
