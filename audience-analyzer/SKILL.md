# Audience Analyzer - 用户画像分析

## 触发场景

- "我的粉丝是什么人"
- "帮我分析一下用户画像"
- "目标用户是谁"
- "用户喜欢什么内容"

## 分析维度

### 1. 基础画像
- 性别分布
- 年龄段分布
- 地域分布
- 设备类型

### 2. 活跃行为
- 活跃时间段
- 活跃日期
- 内容互动时间偏好

### 3. 点赞内容 内容偏好
-类型
- 评论话题方向
- 收藏内容分类
- 转发分享意愿

### 4. 消费能力
- 价格敏感度
- 消费频次
- 客单价区间
- 购买决策周期

### 5. 兴趣标签
- 关注领域
- 搜索习惯
- 行为轨迹
- 生命周期阶段

## 输出格式

```markdown
## 用户画像报告

### 基础属性
- 性别：女性75%，男性25%
- 年龄：25-35岁占比60%
- 地域：一二线城市80%

### 行为特征
- 活跃时间：晚间20:00-22:00
- 活跃周期：周末高于工作日

### 兴趣偏好
- TOP3兴趣：美妆、护肤、时尚
- 内容偏好：教程类、测评类

### 消费画像
- 价格区间：100-300元
- 决策周期：1-3天
```

## 使用方法

1. 提供平台账号
2. 说明分析目的
3. 指定分析维度（可选）
4. 可提供历史数据文件

## 优化建议

根据分析结果给出：
- 内容方向调整建议
- 发布时间优化建议
- 变现模式建议
- 粉丝增长策略
