# Brand Match - 品牌合作匹配

## 触发场景

- "这个品牌适合我吗"
- "合作报价多少合适"
- "品牌匹配度分析"
- "合作风险评估"

## 功能模块

### 1. 匹配度分析
- 品牌调性契合
- 目标人群重叠
- 内容风格匹配
- 价值观一致性

### 2. 报价建议
- 粉丝量级基准
- 内容形式报价
- 行业参考价
- 阶梯报价方案

### 3. 合作模式
- 植入广告
- 定制内容
- 直播带货
- 长期代言

### 4. 风险评估
- 品牌口碑风险
- 内容合规风险
- 粉丝接受度
- 翻车概率评估

## 报价参考公式

```
基础报价 = 粉丝数 × 0.1-0.3元

调整系数：
- 内容质量：高=1.5倍
- 互动率：高=1.3倍
- 垂直度：高=1.2倍
- 原创度：高=1.2倍
- 配合度：高=1.1倍
```

## 输出格式

```markdown
## 品牌合作评估：[品牌名称]

### 匹配度分析
- 调性契合：90%
- 人群重叠：85%
- 综合评分：88%

### 报价建议

- 单条视频：10000-15000元
- 长期合作：月付8折

### 风险评估
- 风险等级：低
- 建议：可以合作

### 注意事项
1. xxx
2. xxx
```

## 使用方法

1. 提供品牌信息
2. 说明合作形式
3. 提供历史数据
4. 获取评估报告
