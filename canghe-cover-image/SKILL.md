# Canghe Cover Image - 封面图生成

## 触发场景

- "做个封面图"
- "生成封面"
- "封面设计"
- "推荐封面风格"

## 平台适配

### 小红书封面
- 比例：3:4（1080×1440）
- 风格：清新、活泼、实用
- 元素：标题大字、产品图、装饰元素

### 公众号封面
- 比例：2.35:1（900×383）
- 风格：简约、大气、专业
- 元素：标题、副标题、品牌色

### 抖音封面
- 比例：9:16（1080×1920）
- 风格：悬念、吸睛、情绪
- 元素：大字标题、人物表情、紧迫感

### B站封面
- 比例：16:9（1920×1080）
- 风格：二次元、醒目、信息量
- 元素：标题、人物、弹幕效果

## 封面公式

```
吸引眼球 = 情绪词 + 数字 + 悬念/利益点

模板1：情绪+数字
"绝了！这3个方法让你皮肤变好"

模板2：问题+答案
"为什么你总是睡不着？原因找到了"

模板3：对比+反转
"月薪3000和30000的区别，就在这3点"

模板4：身份+行为
"宝妈在家做这个，月入过万"
```

## 使用方法

1. 提供视频/文章主题
2. 选择平台类型
3. 指定风格/情绪
4. 获取封面方案

## 输出

- 封面标题建议
- 字体/配色建议
- 图片元素建议
- 文案排版建议
