# Canghe Infographic - 信息图生成

## 触发场景

- "做张信息图"
- "数据可视化"
- "科普图设计"
- "知识点图解"

## 信息图类型

### 1. 数据图表
- 柱状图
- 饼图
- 折线图
- 对比图

### 2. 流程图
- 步骤流程
- 决策树
- 时间线
- 关系图

### 3. 知识卡片
- 定义卡片
- 对比卡片
- 步骤卡片
- 清单卡片

### 4. 科普图解
- 原理图解
- 结构分解
- 对比分析
- 知识图谱

## 设计原则

### 1. 信息层次
- 标题突出
- 核心信息放大
- 辅助信息缩小
- 视觉引导清晰

### 2. 配色方案
- 主色不超过3种
- 对比色突出重点
- 渐变增加层次
- 饱和度适中

### 3. 字体规范
- 标题：粗体、大字号
- 正文：清晰、可读
- 注释：小字号、灰色

## 常用模板

### 对比图
```
┌─────────────────┬─────────────────┐
│     方案A       │     方案B       │
│                 │                 │
│  ✓ 优点1       │  ✓ 优点1       │
│  ✓ 优点2       │  ✓ 优点2       │
│  ✗ 缺点1       │  ✗ 缺点1       │
└─────────────────┴─────────────────┘
```

### 步骤图
```
┌─────┐    ┌─────┐    ┌─────┐
│步骤1│ → │步骤2│ → │步骤3│
└─────┘    └─────┘    └─────┘
```

## 使用方法

1. 提供数据/内容
2. 选择信息图类型
3. 指定风格/配色
4. 获取设计方案

## 输出

- 信息图结构
- 数据可视化建议
- 配色/字体方案
- 制作工具推荐
