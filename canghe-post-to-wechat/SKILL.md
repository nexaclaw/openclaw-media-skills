# Canghe Post to WeChat - 公众号发布

## 触发场景

- "发布公众号"
- "文章排版"
- "推送文章"
- "公众号排版"

## 功能特性

### 1. 智能排版
- Markdown转公众号
- 多种主题样式
- 自动格式化
- 字体大小调整

### 2. 封面图
- 自动提取文章重点
- 生成吸引点击的封面
- 多种尺寸适配

### 3. 预览发送
- 发送到微信预览
- 多账号管理
- 团队协作

### 4. 定时发布
- 定时群发
- 多次推送
- 发布记录

## 使用方法

1. 导入文章（Markdown/链接/本地文件）
2. 选择排版样式
3. 生成封面
4. 预览确认
5. 发送/定时

## 排版模板

- 简约风格
- 学术风格
- 情感风格
- 科技风格

## 示例

```bash
# 导入Markdown并排版
wechat-post --file article.md --template simple --output formatted.html

# 发送到微信预览
wechat-post --preview --file formatted.html
```
