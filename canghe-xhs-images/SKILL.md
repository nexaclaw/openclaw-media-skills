# Canghe XHS Images - 小红书图文排版

## 触发场景

- "做小红书笔记"
- "生成小红书图文"
- "小红书排版"

## 功能特点

### 1. 图文模板
- 产品种草模板
- 教程步骤模板
- 对比测评模板
- 日常分享模板

### 2. 图片处理
- 图片裁剪（3:4比例）
- 滤镜调色
- 文字贴图
- 表情贴纸

### 3. 排版样式
- 单图/多图布局
- 图片拼接
- 留白设计
- 视觉层次

### 4. 标签生成
- 热门话题推荐
- 关键词标签
- 表情符号
- 位置标记

## 小红书图文模板

```
[封面图：产品图+大字标题]
[图2：产品展示]
[图3：使用场景]
[图4：效果对比]
[图5：购买链接/二维码]

文字部分：

正文：3-5段，每段50字以内

```

