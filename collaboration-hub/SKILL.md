# Collaboration Hub - 团队协作

## 触发场景

- "团队任务管理"
- "分配任务"
- "查看进度"
- "协作流程优化"

## 功能模块

### 1. 任务管理
- 任务创建与分配
- 截止日期设置
- 优先级排序
- 任务状态追踪

### 2. 进度追踪
- 看板视图
- 时间线视图
- 进度百分比
- 逾期预警

### 3. 成员管理
- 角色权限设置
- 工作量统计
- 绩效追踪
- 技能标签

### 4. 协作流程
- 审批流程
- 评论反馈
- 文件共享
- @提醒

## 内容生产流程

```
选题策划 → 文案撰写 → 素材制作 → 审核修改 → 发布上线

角色分工：
- 策划：选题、大纲
- 文案：撰写、修改
- 设计：配图、排版
- 运营：发布、互动
```

## 任务模板

```markdown
## 任务：[标题]

### 基本信息
- 负责人：@xxx
- 截止日期：2024-03-15
- 优先级：高
- 状态：进行中

### 任务描述
[详细说明]

### 子任务
- [ ] 子任务1
- [ ] 子任务2
- [ ] 子任务3

### 关联资源
- 素材链接：xxx
- 参考文档：xxx

### 备注
[其他信息]
```

## 使用方法

1. 创建团队项目
2. 添加成员角色
3. 分配任务
4. 追踪进度
