# Comment Manager - 评论管理

## 触发场景

- "帮我看看评论"
- "这些评论怎么回复"
- "用户反馈了什么问题"
- "评论数据分析"

## 功能模块

### 1. 评论聚合
- 多平台评论汇总
- 实时评论抓取
- 评论筛选过滤
- 重点评论标记

### 2. 情感分析
- 正面评论识别
- 负面评论预警
- 中性评论归类
- 情绪趋势分析

### 3. 智能回复
- 回复建议生成
- 个性化回复模板
- 敏感评论处理
- 官方回复引导

### 4. 数据统计
- 评论数量趋势
- 热门评论排行
- 评论互动率
- 用户问题汇总

## 回复策略

### 正面评论
- 感谢支持
- 适当互动
- 引导关注

### 负面评论
- 态度诚恳
- 解决问题
- 私信跟进

### 疑问咨询
- 专业解答
- 提供方案
- 引导详情

### 恶意评论
- 不回应
- 拉黑处理
- 举报屏蔽

## 输出格式

```markdown
## 评论分析报告

### 今日评论概况
- 总评论：50
- 正面：40（80%）
- 中性：8（16%）
- 负面：2（4%）

### 热门评论
1. "产品真的好用" - 点赞20

### 用户问题汇总
1. xxx（出现5次）
2. xxx（出现3次）

### 待回复列表
- @用户1：xxx
- @用户2：xxx
```

## 使用方法

1. 提供评论数据/链接
2. 指定分析范围
3. 设定回复风格
4. 生成回复建议
