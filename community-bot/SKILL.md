# Community Bot - 社群运营机器人

## 触发场景

- "帮我管理社群"
- "自动欢迎新成员"
- "社群活跃度太低"
- "定时发送内容"

## 功能模块

### 1. 欢迎新人
- 新成员入群欢迎
- 自我介绍模板
- 群规自动推送
- 引导资料分发

### 2. 内容运营
- 定时内容推送
- 每日分享素材
- 热点话题引导
- 互动活动组织

### 3. 活跃度管理
- 定时@互动
- 话题抛砖引玉
- 投票/问卷收集
- 红包/福利活动

### 4. 数据统计
- 入群/退群记录
- 发言活跃度排行
- 内容互动数据
- 用户留存分析

### 5. 自动回复
- 关键词回复
- 常见问题解答
- 私聊自动回复
- 敏感词过滤

## 社群运营SOP

```
早上9:00 - 早安问候+今日话题
中午12:00 - 午间内容分享
下午3:00 - 互动问答/投票
晚上8:00 - 干货分享
晚上10:00 - 今日小结+明日预告
```

## 使用方法

1. 说明社群平台（微信群/知识星球）
2. 设定运营目标
3. 配置自动化规则
4. 设定发布时间

示例：
"帮我设置一个微信群的自动欢迎语，新人进来后自动发群规"
"每天晚上8点定时分享一条干货知识到社群"
