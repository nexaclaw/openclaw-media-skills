# Competitor Analysis - 竞品分析

## 触发场景

- "帮我分析一下竞品"
- "这个账号为什么做得好"
- "竞品最近在发什么"
- "对手的数据怎么样"

## 分析维度

### 1. 账号基础分析
- 账号定位
- 粉丝数量及增长趋势
- 内容更新频率
- 运营时长

### 2. 内容分析
- 内容类型分布（图/文/视频）
- 内容主题方向
- 更新时间规律
- 内容风格特点

### 3. 数据表现
- 平均点赞/评论/收藏
- 爆款率
- 互动率
- 增长效率

### 4. 爆款拆解
- 爆款内容特征
- 选题方向分析
- 标题套路总结
- 内容结构分析

### 5. 策略洞察
- 变现模式
- 合作资源
- 差异化优势
- 机会点识别

## 输出格式

```markdown
## 竞品分析报告：[账号名称]

### 基础数据
- 粉丝：50万
- 点赞总量：200万
- 更新频率：日更

### 内容分析
- 内容类型：70%视频，30%图文
- 更新时间：每天20:00

### 爆款分析
- 最高点赞：10万+
- 爆款主题：xxx

### 借鉴建议
1. 内容方向：xxx
2. 发布时间：xxx
3. 差异化机会：xxx
```

## 使用方法

1. 提供竞品账号（名称/链接）
2. 指定分析深度（快速/详细）
3. 关注维度偏好
4. 输出报告形式

示例：
"帮我分析小红书上'XX'这个账号的运营策略"
"最近有哪些竞品出了爆款，分析一下原因"
