# Content Performance - 内容表现分析

## 触发场景

- "这篇内容数据怎么样"
- "帮我分析一下这篇笔记"
- "为什么这篇爆了"
- "内容数据复盘"

## 分析维度

### 1. 基础数据
- 阅读/播放量
- 点赞数
- 评论数
- 收藏数
- 转发/分享数
- 关注转化数

### 2. 互动指标
- 点赞率 = 点赞/阅读
- 评论率 = 评论/阅读
- 收藏率 = 收藏/阅读
- 转发率 = 转发/阅读
- 完播率（视频）

### 3. 传播分析
- 首发流量
- 推荐流量
- 搜索流量
- 分享流量
- 付费流量

### 4. 内容特征
- 选题类型
- 标题特征
- 内容结构
- 发布时间
- 标签话题

### 5. 竞品对比
- 同类内容对比
- 行业平均对比
- 历史数据对比

## 爆款特征拆解

```
爆款 = 选题痛点 × 标题吸引力 × 内容质量 × 传播时机

选题痛点：目标用户关心什么问题
标题吸引力：好奇心/利益点/情绪共鸣
内容质量：信息价值/实用程度/情感共鸣
传播时机：热点/节日/平台流量倾斜
```

## 输出格式

```markdown
## 内容表现分析报告

### 基础数据
- 阅读：10万
- 点赞：5000
- 评论：200
- 收藏：3000
- 转发：500

### 互动指标
- 点赞率：5%（高于平均3%）
- 收藏率：3%（高于平均1.5%）
- 互动率：8%

### 数据解读
- 收藏率高说明实用性强
- 转发率较高说明有传播价值

### 优化建议
1. 选题方向：保持当前方向
2. 标题优化：增加情绪词
3. 内容结构：增加互动引导
```

## 使用方法

1. 提供内容链接/数据
2. 指定分析维度
3. 对比基准选择
4. 输出优化建议
