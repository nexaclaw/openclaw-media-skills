# Copywriter - 专业文案写作

## 触发场景

- 用户要求"写一篇文案"
- 需要"种草推荐"、"产品测评"、"品牌故事"
- 需要"写一段推广文案"、"写个卖点文案"
- 小红书笔记、公众号文章、微博文案创作

## 文案类型

### 1. 种草推荐文案
- 开头：痛点引入 + 产品介绍
- 中间：使用体验 + 效果描述 + 真实案例
- 结尾：引导互动（评论、收藏）

### 2. 产品测评文案
- 客观描述产品外观、功能
- 对比竞品分析优缺点
- 使用场景展示
- 总结推荐指数

### 3. 品牌故事文案
- 品牌理念传达
- 创始人/产品背后的故事
- 价值观输出
- 情感共鸣结尾

### 4. 卖点提炼文案
- 核心卖点 + 佐证
- 差异化优势
- 用户利益点
- 行动号召

### 5. 活动宣传文案
- 活动时间/地点/参与方式
- 活动亮点/福利
- 紧迫感/稀缺感
- 二次传播引导

## 输出格式

根据不同平台调整：

**小红书风格**：
- 标题党 + emoji + 话题标签
- 段落简洁，多用换行
- 结尾引导互动

**公众号风格**：
- 标题吸睛
- 结构清晰，层次分明
- 金句点睛

**微博风格**：
- 140字内精华
- 话题标签 #
- 多图/视频配合

## 使用方法

直接告诉 AI：
- 产品/服务是什么
- 目标平台
- 目标人群
- 期望风格
- 字数要求（可选）

示例：
"帮我写一篇小红书种草文案，推荐一款咖啡机，适合上班族使用，风格轻松有趣"
