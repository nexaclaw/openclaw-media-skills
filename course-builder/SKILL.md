# Course Builder - 知识付费课程

## 触发场景

- "帮我设计一门课程"
- "课程大纲怎么写"
- "知识付费怎么定价"
- "课程内容规划"

## 课程类型

### 1. 技能类课程
- 职场技能
- 专业技能
- 生活技能
- 创作技能

### 2. 认知类课程
- 行业洞察
- 思维方法
- 个人成长
- 创业经验

### 3. 实操类课程
- 软件教学
- 操作演示
- 案例实操
- 项目实战

## 课程结构模板

```
## 课程名称：[主题]

### 课程定位
- 目标人群：
- 解决问题：
- 预期效果：

### 课程大纲
第一章：基础入门（3节）
  1.1 概念认知
  1.2 核心原理
  1.3 快速上手

第二章：核心技能（5节）
  2.1 xxx
  ...

第三章：进阶实战（5节）
  ...

第四章：案例拆解（3节）
  ...

### 课程时长
- 总课时：16节
- 每节时长：15-20分钟
- 总时长：约5小时

### 定价策略
- 早鸟价：xxx
- 正价：xxx
- 会员价：xxx
```

## 定价公式

```
定价 = 目标收入 / 预估销量

参考因素：
- 内容价值
- 市场行情
- 品牌溢价
- 促销策略
```

## 使用方法

1. 说明课程主题
2. 提供目标人群
3. 指定课时要求
4. 获取课程方案
