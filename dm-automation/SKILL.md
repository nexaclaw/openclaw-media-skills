# DM Automation - 私信自动化

## 触发场景

- "私信太多了，帮我处理"
- "设置自动回复"
- "粉丝分层管理"
- "关键词自动回复"

## 功能模块

### 1. 自动回复
- 首次关注回复
- 关键词触发回复
- 常见问题自动答
- 营业时间外回复

### 2. 粉丝分层
- 高价值用户标签
- 活跃度分层
- 消费能力分级
- 兴趣标签分类

### 3. 智能分流
- 客服问题转接
- 商务合作标注
- 售后问题标记
- 普通咨询回复

### 4. 数据统计
- 私信数量统计
- 回复效率分析
- 用户问题TOP榜
- 转化追踪

## 自动回复模板

```
首次关注：
"欢迎关注！我是xxx，专注xxx领域。
有任何问题随时私信我~
回复【资料】领取免费资料包"

关键词触发：
- 回复"价格"：发送价格表
- 回复"合作"：发送合作须知
- 回复"课程"：发送课程介绍

营业外时间：
"感谢您的私信！
我会在工作时间尽快回复您（9:00-18:00）
紧急事项请添加微信：xxx"
```

## 使用方法

1. 设置自动回复规则
2. 配置关键词触发
3. 建立粉丝标签体系
4. 定期优化回复内容
