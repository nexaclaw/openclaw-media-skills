# Content Patterns

## Use These Page Roles

- `cover`: stop the scroll with a single promise.
- `problem`: name a pain point, mistake, or common frustration.
- `method`: explain one method, step, or template.
- `checklist`: present `3-5` fast, scannable points.
- `cta`: close with save/share/follow/download behavior.

## Compress The Source Copy

Turn draft text into card text with these limits:

- Cover title: `8-18` Chinese characters total, usually `1-2` lines.
- Subtitle: `10-24` Chinese characters.
- Highlight chip: `4-10` Chinese characters.
- Bullet line: roughly `8-18` Chinese characters.
- One page: one promise, one step, or one checklist.

If the source is long:

1. Mark the single strongest outcome.
2. Remove background explanations first.
3. Convert paragraphs into title, chips, and bullets.
4. Move nuance into later pages instead of shrinking font size.

## Use Working Headline Formulas

These patterns are reliable for Xiaohongshu text cards:

- Audience + result
  - `普通人也能写出会被收藏的标题`
- Number + payoff
  - `3 个公式写出更像爆款的封面文案`
- Mistake + fix
  - `不要再堆形容词了，先写结果`
- Time + outcome
  - `10 分钟搭出一套能反复用的图文模板`
- Checklist + promise
  - `小红书封面不出错的 5 个排版检查点`

## Split A Carousel Predictably

Use this structure when the user gives only a topic or a long draft:

1. Cover: hook + tag + CTA.
2. Page 2: problem or misconception.
3. Page 3: method or step 1.
4. Page 4: method or step 2 / checklist.
5. Last page: summary + save/share/follow CTA.

For longer sets, add one page per method. Do not let one page carry multiple unrelated tips.

## Rewrite Example

Raw draft:

`很多人写小红书标题时会先写背景、再写过程，结果用户在第一眼看不到收益，所以点击率和收藏率都不高。`

Card-ready version:

- Cover title: `先写结果，标题才会被点开`
- Highlight chips: `点击更高` `收藏更多`
- Slide bullets:
  - `第一眼先给收益`
  - `过程放到第二句`
  - `一个标题只保留一个重点`

## Keep The Tone Native To Xiaohongshu

- Prefer spoken Chinese over formal report style.
- Prefer direct payoff words such as `省时间`, `更清楚`, `不踩坑`, `直接套用`.
- Prefer short, concrete language over abstract brand language.
- Avoid over-promising, fake urgency, or dense marketing copy.
