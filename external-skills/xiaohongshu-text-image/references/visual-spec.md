# Visual Spec

## Use These Defaults

- Default ratio: `3:4`
- Default size: `1080x1440`
- Safe margin: `72-96px`
- Title line count: `1-2`
- Body bullet count: `3-5`
- Accent colors per card: `1`

## Keep The Hierarchy Obvious

Order information by visual weight:

1. Title
2. Tag or step label
3. Highlight chips
4. Bullet lines
5. Footer CTA

If two elements feel equally loud, reduce one of them. Xiaohongshu cards work best when the eye knows where to land immediately.

## Protect Readability

- Keep long paragraphs out of the image.
- Use large line height for Chinese copy.
- Leave empty space around the title block.
- Avoid placing highlight chips too close to the footer.
- Use contrast through size and weight before adding more colors.

## Choose A Layout Family

### Warm Note

- Light background
- Dark text
- Soft warm accent
- Useful for growth, productivity, study, and lifestyle content

### Sharp Contrast

- Light neutral background
- Strong dark text
- Saturated red or orange accent
- Useful for checklist, mistakes, before/after, and punchier hooks

## Use Accent Sparingly

Accent is for one of these jobs:

- the tag
- the most important words in the hook
- highlight chips
- the final CTA

Do not color every keyword. If more than roughly `10-15%` of the visible text uses the accent color, the page usually becomes noisy.

## Match Layout To Content

- Use `cover` when the page needs one strong promise.
- Use `slide` when the page explains one step or one cluster of points.
- Use `quote` when a single sentence is the whole value.
- Use `checklist` when the user wants a practical saveable asset.

## Prefer System Fonts

Use common Chinese-capable font families in this order unless the user provides a specific brand font:

`PingFang SC, Hiragino Sans GB, Microsoft YaHei, sans-serif`
