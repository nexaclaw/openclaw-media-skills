# Feedback Collector - 反馈收集

## 触发场景

- "用户满意度调查"
- "收集用户反馈"
- "设计问卷"
- "分析用户意见"

## 功能模块

### 1. 问卷设计
- 问题类型选择
- 问卷逻辑设计
- 视觉美化
- 投放渠道设置

### 2. 多渠道收集
- 微信问卷
- 小程序问卷
- 表单工具
- 评论收集
- 私信收集

### 3. 数据分析
- NPS评分分析
- 满意度统计
- 问题分类汇总
- 趋势变化追踪

### 4. 改进建议
- 问题优先级排序
- 改进方案建议
- 执行计划制定

## 问卷模板

```
## 用户满意度调研

感谢您参与本次调研！

1. 您使用我们产品多久了？
   □ 1个月内
   □ 1-3个月
   □ 3-6个月
   □ 6个月以上

2. 您对产品整体满意度？
   ○ 非常满意
   ○ 满意
   ○ 一般
   ○ 不满意
   ○ 非常不满意

3. 您最看重哪些功能？（多选）
   □ 功能A
   □ 功能B
   □ 功能C
   □ 其他____

4. 您有什么建议？
   [开放题]
```

## 使用方法

1. 说明调研目的
2. 提供目标人群
3. 指定问题方向
4. 获取问卷和分析
