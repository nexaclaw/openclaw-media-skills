# Follower Growth - 粉丝增长分析

## 触发场景

- "粉丝增长情况怎么样"
- "帮我分析一下涨粉原因"
- "为什么粉丝在掉"
- "怎么才能涨粉"

## 分析维度

### 1. 增长趋势
- 日/周/月增长曲线
- 增长速率
- 增长拐点
- 周期性规律

### 2. 增长来源
- 推荐页流量
- 搜索流量
- 分享流量
- 付费流量
- 直播转化

### 3. 流失分析
- 流失数量
- 流失时间点
- 流失用户特征
- 流失原因推测

### 4. 增长策略
- 优质内容驱动
- 互动引流策略
- 跨平台导流
- 付费推广优化
- 爆款内容复制

## 输出格式

```markdown
## 粉丝增长报告

### 整体趋势
- 现有粉丝：10万
- 本月增长：+5000
- 增长率：5%

### 来源分析
- 推荐页：60%
- 搜索：20%
- 分享：15%
- 其他：5%

### 流失分析
- 本月流失：-200
- 流失率：0.2%

### 增长建议
1. 保持日更频率
2. 增加互动内容
3. 优化发布时间
```

## 使用方法

1. 提供账号数据
2. 指定分析周期
3. 关注特定问题
4. 定制优化建议
