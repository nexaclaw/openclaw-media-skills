# Multi-Post - 多平台分发

## 触发场景

- "帮我把内容发到所有平台"
- "这篇内容同步到抖音和小红书"
- "多平台一键发布"
- "内容分发到各个账号"

## 支持平台

- 小红书
- 微信公众号
- 抖音
- 微博
- B站
- 知乎
- 今日头条

## 功能特点

### 1. 格式自动适配
- 标题长度调整
- 内容结构转换
- 表情符号适配
- 标签话题替换

### 2. 平台差异化
- 小红书：emoji+话题+图片
- 公众号：严谨+分段+金句
- 抖音：口语+悬念+热梗
- 微博：简短+话题+@

### 3. 发布时间优化
- 各平台黄金时段推荐
- 错峰发布策略
- 定时发布设置
- 发布间隔建议

### 4. 发布状态追踪
- 各平台发布结果
- 发布链接汇总
- 失败原因记录
- 重新发布机制

## 工作流程

```
1. 输入原始内容（文档/链接/主题）
2. 选择目标平台
3. 自动适配各平台格式
4. 确认发布（预览/修改）
5. 一键分发
6. 汇总发布结果
```

## 使用方法

1. 提供原始内容
2. 选择发布平台
3. 设置发布时间（立即/定时）
4. 确认发布

示例：
"把这篇关于春季穿搭的文章同步发到小红书和公众号"
"帮我把这条视频内容发布到抖音、B站和微博"
