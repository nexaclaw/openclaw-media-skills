# QR Generator - 品牌二维码生成

## 触发场景

- "生成二维码"
- "做一个带Logo的二维码"
- "品牌二维码"
- "二维码美化"

## 功能特性

### 1. 基础二维码
- 网址二维码
- 文本二维码
- WiFi二维码
- 名片二维码

### 2. 品牌定制
- Logo嵌入
- 品牌配色
- 自定义图案
- 样式模板

### 3. 数据追踪
- 扫描统计
- 来源分析
- 地域分布
- 时间趋势

### 4. 输出格式
- PNG高清
- SVG矢量
- PDF印刷版

## 使用方法

1. 选择二维码类型
2. 输入内容
3. 选择样式
4. 生成并下载

## 示例

```bash
# 生成基础二维码
qr-generator --content "https://example.com" --output qr.png

# 带Logo的二维码
qr-generator --content "https://example.com" --logo logo.png --style brand
```
