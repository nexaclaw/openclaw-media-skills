# Script Writer - 短视频脚本写作

## 触发场景

- "帮我写个视频脚本"
- "口播脚本怎么写"
- "帮我写一段台词"
- "视频脚本模板"

## 脚本类型

### 1. 口播脚本
- 人物出镜说话
- 逻辑清晰
- 情绪表达
- 节奏把控

结构：
```
开场钩子（3秒）
自我介绍（3秒）
核心内容（60-90秒）
总结金句（10秒）
行动引导（5秒）
```

### 2. 剧情脚本
- 人物设定
- 情节推进
- 对话设计
- 情绪转折

结构：
```
场景1：开场（冲突引入）
场景2：发展（矛盾升级）
场景3：高潮（冲突爆发）
场景4：结尾（反转/升华）
```

### 3. 测评脚本
- 产品介绍
- 体验过程
- 优缺点总结
- 推荐结论

结构：
```
开场：产品亮相+亮点剧透
外观：开箱+第一印象
体验：核心功能演示
对比：vs竞品
总结：适合谁+评分
```

### 4. Vlog脚本
- 日常记录
- 主题明确
- 节奏轻快
- 情感真实

结构：
```
开场：今日主题预告
正文：场景1-场景2-场景3
结尾：今日感悟+明日计划
```

## 脚本格式

```markdown
# [视频标题]

## 基本信息
- 时长：2分钟
- 类型：口播
- 风格：轻松有趣

## 脚本内容

| 时长 | 画面 | 台词 | 备注 |
|------|------|------|------|
| 0:03 | 镜头特写 | 这样的手机壳，99%的人都没见过 | 抛出悬念 |
| 0:10 | 人物出场 | 哈喽大家好，我是xxx | 自我介绍 |
...
```

## 使用方法

1. 说明视频主题
2. 选择脚本类型
3. 指定时长和风格
4. 提供产品/内容信息

示例：
"帮我写一个2分钟的口播脚本，推荐一款护肤品"
"一个剧情反转的小视频，大概30秒"
