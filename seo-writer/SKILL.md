# SEO Writer - SEO长文写作

## 触发场景

- "帮我写一篇SEO文章"
- "优化这篇文章的关键词"
- "这个关键词怎么布局"
- "帮我做关键词分析"

## 核心功能

### 1. 关键词研究
- 主关键词确定
- 长尾关键词挖掘
- 关键词竞争度分析
- 搜索意图分析

### 2. 内容结构优化
- 标题包含主关键词
- H1/H2/H3层级清晰
- 段落长度适中（150-300字）
- 内链外链布局

### 3. 关键词布局策略
- 标题：主关键词前置
- 首段：主关键词出现1次
- 正文：关键词密度2-5%
- 小标题：长尾关键词
- 结尾：主关键词呼应

### 4. 内容质量要求
- 原创性 > 80%
- 内容深度 > 竞品
- 用户价值明确
- 信息更新及时

## 文章模板

```markdown
# [主关键词]完整指南：[吸引点击的副标题]

## 什么是[主关键词]？
[定义+重要性，包含关键词]

## 为什么[主关键词]很重要？
[痛点分析+价值说明]

## [主关键词]的5个核心方法
### 方法一：[长尾关键词1]
### 方法二：[长尾关键词2]
...

## 常见问题解答
Q: [长尾关键词问题]？
A: [详细解答]

## 总结
[要点回顾+行动号召]
```

## 平台适配

- **公众号**：标题吸引人，正文深度
- **知乎**：问答形式，专业权威
- **博客/网站**：技术导向，数据支撑
- **小红书**：关键词标签化，信息密度

## 使用方法

1. 提供主关键词
2. 说明目标平台
3. 指定字数要求
4. 提供竞品链接（可选）

示例：
"写一篇关于'减肥食谱'的SEO文章，目标知乎，3000字左右"
