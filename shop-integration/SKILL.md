# Shop Integration - 小店运营

## 触发场景

- "帮我管理小店"
- "商品上架"
- "订单处理"
- "库存预警"

## 支持平台

- 抖音小店
- 快手小店
- 小红书店铺
- 微信小商店
- 淘宝店铺

## 功能模块

### 1. 商品管理
- 批量上架/下架
- 商品信息同步
- 价格批量调整
- 商品分类管理

### 2. 订单管理
- 订单状态追踪
- 异常订单预警
- 自动发货设置
- 退换货处理

### 3. 库存管理
- 实时库存同步
- 低库存预警
- 补货提醒
- 库存报表

### 4. 售后管理
- 客服消息汇总
- 退款申请处理
- 差评预警
- 客户满意度追踪

## 运营SOP

```
每日：
- 查看昨日订单
- 处理待发货订单
- 回复客服消息
- 检查库存预警

每周：
- 分析销售数据
- 优化商品信息
- 调整价格策略
- 处理售后问题
```

## 使用方法

1. 绑定店铺账号
2. 设置管理规则
3. 配置自动化
4. 查看运营报表
