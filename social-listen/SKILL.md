# Social Listen - 社交监听

## 触发场景

- "品牌舆情监控"
- "用户提到了什么"
- "行业动态追踪"
- "竞品动向分析"

## 监听维度

### 1. 品牌提及
- 品牌名称/产品名
- 正负面情感分析
- 提及平台分布
- 影响力评估

### 2. 用户反馈
- 产品评价收集
- 服务投诉监测
- 改进建议汇总
- 用户需求洞察

### 3. 行业动态
- 行业热点追踪
- 政策法规更新
- 技术趋势分析
- 市场机会识别

### 4. 竞品动向
- 竞品动态监控
- 新品发布追踪
- 营销活动分析
- 用户反馈对比

## 监听平台

- 微博
- 小红书
- 抖音
- 知乎
- 微信公众号
- 贴吧/论坛
- 新闻媒体

## 输出格式

```markdown
## 舆情日报：[日期]

### 品牌提及概况
- 总提及：50次
- 正面：40次（80%）
- 中性：8次（16%）
- 负面：2次（4%）

### 热点话题
1. xxx（微博热搜）
2. xxx（小红书热门）

### 竞品动态
- 竞品A：发布新品xxx
- 竞品B：启动活动xxx

### 预警事项
- 负面舆情：xxx（需关注）
```

## 使用方法

1. 设置监听关键词
2. 选择监听平台
3. 设定预警阈值
4. 定期查看报告
