# Subtitle Generator - 字幕生成工具

## 触发场景

- "生成字幕"
- "视频翻译"
- "添加字幕"
- "字幕校对"

## 功能特性

### 1. 语音转字幕
- 自动语音识别（ASR）
- 支持中文、英文、日文等
- 准确率优化

### 2. 多语言翻译
- 原文翻译
- 人工校对模式
- 术语库支持

### 3. 字幕编辑
- 时间轴调整
- 批量替换
- 样式设置

### 4. 输出格式
- SRT字幕文件
- VTT网页字幕
- ASS特效字幕
- 纯文本

## 使用方法

1. 上传音视频
2. 选择语言
3. 生成字幕
4. 校对编辑
5. 导出

## 示例

```bash
# 语音转字幕
subtitle-generator --video input.mp4 --lang zh-CN --output subtitles.srt

# 翻译字幕
subtitle-generator --srt original.srt --translate en --output english.srt
```
