# Topic Generator - 选题生成器

## 触发场景

- "今天发什么内容"
- "帮我生成一些选题"
- "有什么热门话题可以写"
- "给小红书推荐几个选题"

## 选题来源

### 1. 热点追踪
- 微博热搜
- 抖音热榜
- 小红书热门
- 知乎热榜
- 百度指数

### 2. 用户需求
- 搜索词分析
- 评论区问题收集
- 私信咨询汇总
- 竞品爆款拆解

### 3. 节日节点
- 公历节日
- 农历节日
- 行业节点
- 纪念日

### 4. 垂直领域
- 行业新闻
- 技术更新
- 政策解读
- 趋势分析

## 选题公式

1. **热点+观点**：某事件 + 独特角度解读
2. **痛点+方案**：用户问题 + 解决方法
3. **对比+分析**：A vs B + 优劣对比
4. **案例+复盘**：成功案例 + 可复制方法
5. **盘点+推荐**：Top10盘点 + 选择建议
6. **预测+趋势**：行业分析 + 未来展望

## 输出格式

每个选题包含：
- 选题标题
- 内容类型（图文/视频/直播）
- 目标平台
- 目标人群
- 预估热度
- 执行难度
- 内容大纲

## 使用方法

1. 提供账号定位/领域
2. 说明目标平台
3. 指定选题数量
4. 可选：指定选题方向

示例：
"我是做美妆的，帮我生成10个小红书选题，偏向护肤类"
"结合今天的热点，给科技公众号推荐5个选题"
