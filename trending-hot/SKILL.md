# Trending Hot - 全平台热搜监控

## 触发场景

- "今天有什么热点"
- "查看微博热搜"
- "抖音热门话题"
- "最近什么火"
- "竞品热度"

## 支持平台

- 微博热搜
- 抖音热榜
- 小红书热门
- B站热搜
- 知乎热榜
- 百度热搜
- 腾讯新闻

## 功能特性

### 1. 实时热搜监控
- 定时抓取各平台热搜榜单
- 支持 API 对接和网页爬取
- 热度和排名变化追踪

### 2. 关键词追踪
- 自定义监控关键词
- 多关键词组合
- 实时推送提醒

### 3. 热点分析
- 热度趋势图
- 话题生命周期
- 关联话题挖掘

### 4. 选题推荐
- 基于热点生成选题建议
- 热度预测
- 竞争度分析

## 输出格式

```markdown
## 今日热点速览

### 🔥 微博 TOP5
1. #话题1# - 热度 500w+
2. #话题2# - 热度 300w+
...

### 📱 抖音 TOP5
1. 话题1 - 播放 1000w+
...

### 💡 选题建议
- 角度1：结合热点话题X
- 角度2：蹭热点话题Y
```

## 使用方法

1. 配置监控平台
2. 设置关键词
3. 获取实时热点
