# Video Editor - 视频剪辑助手

## 触发场景

- "帮我规划视频剪辑"
- "这个视频怎么剪"
- "镜头顺序怎么安排"
- "配什么音乐好"

## 功能模块

### 1. 剪辑建议
- 素材分析评估
- 保留/删除建议
- 节奏把控
- 重点片段标记

### 2. 分镜规划
- 场景划分
- 镜头顺序
- 时长分配
- 转场建议

### 3. 节奏优化
- 节奏点标记
- 音乐卡点
- 情绪曲线
- 高潮设计

### 4. 字幕匹配
- 字幕位置
- 字体样式
- 动画效果
- 关键词高亮

### 5. 音乐推荐
- 风格匹配
- 节奏参考
- 版权检查
- 音量调整

## 视频结构模板

```
开头（3秒）：
- 钩子：吸引眼球的画面/问题

正文（60秒）：
- 痛点引入（10秒）
- 解决方案（20秒）
- 案例展示（15秒）
- 总结要点（15秒）

结尾（5秒）：
- 行动号召
- 关注引导
```

## 使用方法

1. 提供视频素材/主题
2. 说明目标平台
3. 指定视频时长
4. 描述期望风格

示例：
"帮我规划一个3分钟的知识科普视频怎么剪"
"这段采访素材太多了，帮我看看怎么精简"
