# Weather - 天气查询

## 触发场景

- "今天天气怎么样"
- "明天会下雨吗"
- "查看天气"
- "北京天气"

## 功能特性

### 1. 实时天气
- 当前温度
- 天气状况（晴/雨/阴等）
- 湿度
- 风力
- 空气质量

### 2. 天气预报
- 未来24小时
- 未来7天
- 温度趋势
- 降水概率

### 3. 城市查询
- 国内城市
- 国际城市
- 自动定位

## 使用方法

```
用户：北京今天天气
AI：今天北京晴，25°C，空气质量良...

用户：上海明天会下雨吗
AI：明天上海小雨，气温20-26°C...
```

## 数据来源

- 中国天气网
- 或其他免费天气API
